# Greenfield University Website

A basic university website with an online recruitment / admissions process.

## Pages

- **index.html** – Landing page with hero banner, programme cards, about section, and contact info.
- **recruitment.html** – Application form with client-side validation for prospective students.

## Features

- Responsive, mobile-friendly layout
- Sticky navigation with hamburger menu on small screens
- Recruitment form with real-time field validation
- Confirmation screen shown on successful submission

## Getting Started

Open `index.html` in any modern web browser — no build step required.
